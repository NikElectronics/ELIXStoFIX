Инструкция по работе с диагностическим модулем аккумуляторов Makita

Общий вид устройства

1. Подключение и настройка

1.1. Аппаратное подключение

  1. Подключите диагностическое устройство к аккумулятору Makita через специальный разъем (рис. 1.1)
  2. Подключите устройство к компьютеру через USB-порт (рис. 1.2)
  3. Включите прибор нажав кнопку включения (рис. 1.1)
  4. Воспользуйтесь программой open battery information
  5. После завершения работы нажмите кнопку включения 2 раза для выключения устройства (рис. 1.1)
Рис. 1.1 - Подключение устройства к аккумулятору BL1860
Рис. 1.2 - Подключение через USB-порт

Внимание! Не подключайте устройство к аккумулятору при включенном питании устройства!

1.2. Настройка Wi-Fi

Устройство автоматически создает защищенную точку доступа Wi-Fi со следующими параметрами:

SSID: Makita

Пароль: 12345678

IP-адрес: 192.168.4.1

После подключения к "Makita" откройте браузер и перейдите по адресу http://192.168.4.1

Рис. 1.4 - Стартовый экран веб-интерфейса

2. Работа с веб-интерфейсом

2.1. Чтение данных

Для получения информации об аккумуляторе выполните следующие действия:

  1. В разделе "📚 Чтение данных" нажмите кнопку "Модель" (рис. 2.1)
  2. Дождитесь определения модели аккумулятора
  3. После успешного определения нажмите кнопку "Данные"
  4. Ожидайте завершения сбора данных
Рис. 2.1 - Определение модели аккумулятора
Рис. 2.2 - Пример вывода данных аккумулятора

Если нет ошибок то в логе отображаются следующие параметры:

  • Напряжение: общее и по каждой ячейке (5 ячеек)
  • Разница напряжений: между максимальной и минимальной ячейкой
  • Температура: элементов и силовых транзисторов

2.2. Управление светодиодами

После успешного определения модели аккумулятора станет доступен раздел "💡 Светодиоды":

Рис. 2.3 - Панель управления светодиодами
  1. Нажмите "Включить" для активации всех светодиодов аккумулятора
  2. Нажмите "Выключить" для деактивации светодиодов

Некоторые модели аккумуляторов могут не поддерживать эту функцию!

2.3. Сброс ошибок

В разделе "▶️ Сброс данных" доступны функции обслуживания:

Рис. 2.4 - Панель сброса ошибок
  1. Нажмите "Ошибки" для сброса ошибок аккумулятора
  2. Дождитесь подтверждения операции

Внимание! Сброс ошибок выполняется только при наличии лицензионного ключа!

3. Программатор

В разделе "👨‍💻 Программатор" доступны дополнительные функции управления устройством:

Рис. 3.1 - Интерфейс программатора
  1. Перезагрузить: полная перезагрузка диагностического устройства
  2. Ключ: поле для ввода лицензионного ключа (предустановлен)

После ввода правильного ключа (указан на корпусе) станут доступны все функции диагностики

4. Зарядка встроенного аккумулятора

Устройство оснащено встроенным аккумулятором c портом для заряда USB Type C:

Рис. 4.1 - Зарядка аккумулятора
  1. Подключите устройство при помощи кабеля к зарядному устройству 5в 1A
  2. Индикатор заряда в процессе будет мигать крассным цветом
  3. Дождитесь завершения процесса (все светодиоды горят зеленым)

Важно!

  • Не оставляйте устройство без контроля во время заряда.
